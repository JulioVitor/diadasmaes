# 💝 Feliz Dia das Mães - Homenagem Especial

Uma página web interativa para homenagear sua esposa no Dia das Mães.

## ✨ Funcionalidades

- 🎨 Modo claro/escuro com persistência
- 🎵 Música de fundo com controle
- ⏰ Contador regressivo para o próximo Dia das Mães
- 📝 Mensagem com efeito máquina de escrever
- 🖼️ Carrossel de fotos
- 📸 Galeria estilo Polaroid
- 🧠 Jogo da memória temático
- ✍️ Assinatura digital touch/mouse
- 📱 QR Code para acesso mobile
- 📤 Compartilhamento no WhatsApp
- 🖨️ Impressão do QR Code
- 🌸 Chuva de pétalas e corações flutuantes
- 🎉 Confetes no carregamento

## 🚀 Como usar

1. Clone o repositório
2. Abra o `index.html` no navegador
3. Personalize as fotos e mensagens
